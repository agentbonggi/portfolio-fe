# Portfolio Frontend

Next.js + TypeScript + Tailwind CSS

## Setup

```bash
npm install
npm run dev
```

## Environment

Create `.env.local`:
```
NEXT_PUBLIC_API_URL=http://localhost:8080/api/v1
```

## Features

- Projects showcase
- Skills display
- Experience timeline
- Contact form
- Dark mode support

## Tech Stack

- Next.js 16 (latest)
- Tailwind CSS 4 (latest)
- TypeScript
- Dark/Light mode via CSS variables

## Theme

Adjust colors in `src/app/globals.css` CSS variables.
